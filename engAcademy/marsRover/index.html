Mars Rover Lab

Semester 1 Midterm

12/18/25

For our midterm, we were tasked with completing a whole of tasks relating to a mars rover. These included moving a reactor core, getting soil samples, and other similar tasks a mars rover might do. We had 10 minutes to complete 8 tasks. We could only see the robot every other minute so we had to create programs for each task to run. Also, because of the intervals when we couldn't see the robot, it was a necessity that each program would work every single time. In the end our rover was very successful.
