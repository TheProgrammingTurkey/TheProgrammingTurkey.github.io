Advent Of Code 2022

First Crack At The Yearly Event

12/25/22

Advent of Code is an Advent calendar of small programming puzzles split in 2 parts, every day for 25 days before Christmas. Any programming language is allowed and I decided to use Javascript. My brother recommended that I tried it and I managed to complete levels 1 through 4, and 6. I absolutely recommend trying this yearly challenge as it definitely improves problem-solving and programming abilities.
