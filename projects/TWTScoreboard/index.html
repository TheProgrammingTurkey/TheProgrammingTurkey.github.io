TWT Scoreboard

Card Game Scoreboard

7/14/22

TWT Scoreboard was a project that I made while on vacation before my freshman year of high school. I was struggling to find any good ideas to code and my brother suggested I should make an online scorer for the card game pounce. While it did start as a pounce scorer we changed to be a fairly universal scorer. This project also improved my Javascript being one of the most complex project in terms of Javascript that I've made.
