Ultimate Tic Tac Toe

Tic Tac Toe within Tic Tac Toe

6/6/26

Ultimate Tic Tac Toe is one of those random childhood games that I remember vividly. Basically, it's a tic tac toe game with a tic tac toe game in each square. The goal is to win the main tic tac toe board by winning the smaller tic tac toe boards. I've always wanted to make this into a website but I didn't want to spend weeks or months with the styling and AI. So instead I used it as an opportunity to see what AI programming is like right now. This entire project was made by Github Copilot, and I was blown away by how well it worked. I would estimate the entire project was made by 15 or so prompts. Feel free to check it out below.
